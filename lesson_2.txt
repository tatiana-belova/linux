Задание 1. Команда cat предназначена для чтения данных из файла и выведения их на экран. 
Может использоваться для объединения файлов.
Чтобы создать файлы мне понадобилась команда cat > new_file. 
Чтобы объеденить файлы я использовала команду cat new_file new_file1 > new_file2
Чтобы скопировать файл с новым именем я использовала cat new_file2 > copy_of_new_file2
Задание 2. Для подсчета количества  скрытых файлов в домашнем каталоге 
я воспользовалась командой ls -А | grep '^\.' |wc
ls - просмотр сожердимого каталога
 -А - отображение всех файлов, включая скрытые, кроме . и ..
 | grep '^\.' - для поиска фалов, нчинающихся на .
 |wc - для подстчета кол-ва строк.
