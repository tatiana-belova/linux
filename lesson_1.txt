Задание 5:
Команда для включения фаерволла - sudo ufw enable, так как для запуска этого сценария требуются права администратора
Чтобы добавить ssh порт TCP 22 в исключения используется команда sudo ufw allow 22/tcp
ssh-keygen - это программа, генерирующая пару ключей - приватный и публичный.
Приватная часть SSH ключа - это секретный ключ, который всегда остается у пользователя и никому не показывается.
Публичная часть SSH ключа копируется на удаленный SSH сервер и хранится в специальном файле, известном SSH серверу. 
Публичная часть SSH ключа используется для шифрования сообщений, которые можно расшифровать только приватной частью ключа
authorized_keys - это файл, в котором хранится публичный ключ.
Клиент может подключаться к удаленной машине с помощью логина (username) и IP-адреса — например, ssh username@123.456.789.00. 
Ключи будут сопоставляться автоматически.
